About

Discord Summarizer is a Chrome extension that helps you catch up on conversations by summarizing unread Discord messages. It uses Google Gemini-2.5-pro to generate concise, accurate summaries of conversations, saving you time and helping you stay in the loop.

Whether you've been away from a busy server or just want to quickly understand what's happening in a channel, Discord Summarizer makes it easy to get up to speed without scrolling through hundreds of messages.

Features

Manual Summarization

Trigger summarization with a single click whenever you need it.

Works Everywhere

Functions across all Discord servers and channels without restrictions.

Multiple Modes

Choose between Brief, Detailed, or Key Takeaways summarization styles.

Customizable Format

Get summaries in paragraph format or as bullet points based on your preference.

Message Selection

Summarize unread messages or choose a specific number of recent messages.

Seamless Integration

Summaries appear directly in Discord's interface for a native experience.

Installation

  1. Clone the repository or download the source code from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top-right corner
  4. Click "Load unpacked" and select the extension folder from the repository
  5. The Discord Summarizer extension should now appear in your extensions list
  6. Pin the extension to your toolbar for easy access

How to Use

  1. Navigate to Discord in your browser
  2. When you see unread messages (indicated by the "New Messages" divider), click the "Summarize" button
  3. Choose your preferred summarization mode from the dropdown (Brief, Detailed, or Key Takeaways)
  4. Select whether to summarize unread messages or a specific number of recent messages
  5. The summary will appear directly in the Discord chat

Technology

Discord Summarizer is built with:

  • JavaScript (Chrome Extension Manifest V3)
  • Node.js/Express (Backend API)
  • Google Gemini-2.5-pro-exp-03-25 (AI Summarization)

The extension communicates with a secure backend that handles the summarization process using Google's advanced AI models.
